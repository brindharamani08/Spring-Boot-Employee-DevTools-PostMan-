EMPLOYEE MANAGEMNET SYSTEM

Project Overview:
The Employee Management System is a Spring Boot REST API project developed to perform CRUD (Create, Read, Update, Delete) operations on employee records. The application follows a layered architecture consisting of Controller, Service, Repository (DAO), and Entity layers.
The project uses Hibernate (JPA) for Object Relational Mapping (ORM) and Maven for dependency management. RESTful APIs were created and tested successfully using Postman.

FEATURES:

Add a new Employee
Get ONE Employees
Get Employee by ID
Delete Employee
RESTful API implementation
Layered Architecture
Hibernate ORM Integration
Maven Project
Spring Boot DevTools support
JSON Request & Response
Postman API Testing
